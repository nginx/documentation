## Overview

Use this reference to look up all user-configurable settings and indicator light states on a standard drip coffee maker. Each settings entry includes the control name, accepted values, the default, and a description of its effect. Each indicator entry describes what the light state means and what action, if any, the user should take.

---

## Brew settings

These settings control how the coffee maker heats water and passes it through the grounds. Configure them before pressing the Brew button.

| Name | Accepted values | Default | Description |
|:--- |:--- |:--- |:--- |
| `Brew strength` | `Regular`, `Bold` | `Regular` | Controls the water flow rate through the filter basket. `Regular` produces a standard-strength cup. `Bold` slows the flow rate, increasing the contact time between water and grounds and producing a stronger cup. |
| `Cup volume` | `1` – `12` (cups) | `4` | Sets the target brew volume in standard cups (180 ml each). The coffee maker stops the brew cycle when the selected volume has passed through the filter basket. Do not set a volume above the water level in the reservoir. |
| `Water temperature` | `Low`, `Medium`, `High` | `High` | Sets the target water temperature at the heating element. `High` (96 °C) is recommended for most ground coffees. Reduce to `Medium` (90 °C) for light roasts or to `Low` (85 °C) for cold-bloom brewing. |

---

## Keep-warm settings

These settings control the warming plate that maintains carafe temperature after the brew cycle completes.

| Name | Accepted values | Default | Description |
|:--- |:--- |:--- |:--- |
| `Keep-warm` | `On`, `Off` | `On` | Enables or disables the warming plate after the brew cycle completes. Set to `Off` if you plan to transfer coffee to a thermal carafe immediately after brewing. |
| `Keep-warm temperature` | `Low`, `High` | `Low` | Sets the warming plate temperature. `Low` (70 °C) slows the rate at which the coffee degrades. `High` (80 °C) maintains a hotter cup but accelerates bitterness in coffee held for more than 30 minutes. |
| `Keep-warm duration` | `30`, `60`, `120` (minutes) | `60` | Sets how long the warming plate stays active after the brew cycle completes. The warming plate turns off automatically when the duration expires. |

---

## Indicator light states

The indicator light on the front of the unit communicates the current operating state of the coffee maker. No user action is required unless noted.

| State | Meaning | Action required |
|:--- |:--- |:--- |
| Solid on | The unit has power and is ready to brew. | None. |
| Slow blink (1 per second) | A brew cycle is active. | None. Wait for the cycle to complete. |
| Fast blink (3 per second) | The water reservoir is empty or the reservoir lid is not fully seated. | Refill the reservoir and press the reservoir lid down until it clicks. |
| Solid off, unit plugged in | The unit has lost power or the keep-warm duration has expired. | Check the outlet. If the outlet has power, press the Brew button to restart. |
