NGINX Instance Manager 2.22.1 release notes
## Upgrade Paths {#2-22-1-upgrade-paths}

NGINX Instance Manager 2.22.1 supports upgrades from these previous versions:

- 2.19.0 - 2.22.0

If your NGINX Instance Manager version is older, you may need to upgrade to an intermediate version before upgrading to the target version.

## What's new {#2-22-1-whats-new}

This release includes the following updates:

- {{% icon-feature %}} **Support added for several Linux distributions**<a name="2-22-1-whats-new-Support-added-for-several-Linux-distributions-47318"></a>

   Added support for the following operating systems:
   - Debian 13
   - Red Hat Enterprise Linux 10
   - Rocky Linux 8
   - Rocky Linux 9
   - Rocky Linux 10

   For a complete list of supported platforms, see the [Supported distributions]({{< ref "/nim/fundamentals/tech-specs.md#supported-distributions" >}}) documentation.

You can find information about known issues in the [Known Issues]({{< ref "/nim/releases/known-issues.md" >}}) topic.
