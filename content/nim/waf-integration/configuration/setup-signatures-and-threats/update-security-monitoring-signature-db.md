The Security Monitoring dashboards in F5 NGINX Instance Manager use an attack signature database to display details about detected security violations, including each signature’s name, accuracy, and risk level.

To ensure your dashboards remain accurate and up to date, update the Security Monitoring attack signature database regularly.
