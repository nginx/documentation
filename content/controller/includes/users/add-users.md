Take the following steps to create a User:

1. Open the NGINX Controller user interface and log in.
2. Select the NGINX Controller menu icon, then select **Platform**.
3. On the **Platform** menu, select **Users**.
4. On the **Users** menu, select **Create User**.
5. Add a first name.
6. Add a last name.
7. Add an email address.
8. Add a password.
9. (Optional) Add one or more tags.
10. (Optional) Add one or more [Roles]({{< relref "/platform/access-management/manage-roles.md" >}}). The Role specifies the user's permission level.
